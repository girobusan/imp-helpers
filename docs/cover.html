Cover Helper

Add longread style cover

Longread-style cover

This helper renders static html and can make some ajustments in interactive mode.

→ See this page in edit mode.

Code

 ```h:cover
 title: Cover Helper
 subtitle: Add longread style cover
 noPreview: true
```

Parameters

Parameters are YAML-formatted.

Parameter Meaning Default value
title Title to display "Title"
Subtitle Subtitle to display ""
height css value — default height of the cover "50vh"
minHeight css value — minimum height "300px"
marginBottom css value — margin below the cover "1rem"
bgColor css value — background color "black"
color color of the text "white"
bgImage url, background image for cover ""
tiled tile background (by default, picture will fill the cover) false
removeH1 Remove first H1 on the page false
noPreview Do not display cover in preview mode false
