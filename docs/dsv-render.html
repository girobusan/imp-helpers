⚙️ dsv-render

DSV format renderer

This helper renders only static html. Just put your DSV code inside fences. See this example in edit mode. It has a subtype no-header for skiping header formatting.

Example, without subtype

Code

 ```h:dsv-render
 Survived,Sex,Age,Class,value
 Perished,Male,Adult,Crew,670
 Perished,Male,Adult,Third Class,387
 Perished,Male,Adult,Second Class,154
 ```
